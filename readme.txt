В этом проекте находятся две программы (Алгоритм Карацюбы и MergeSort). Последний алгоритм реализован немного другим методом, чем в лекции (но "скелет" тот же).
Программки запускаются через main.
Дополнительно реализовано два класса, они нужны для работы с текстовым документом и "красивым" оформлением результата программок.
Чтобы ввести свои значения, достаточно открыть файлы examples и туда внести свои числа через пробел.
